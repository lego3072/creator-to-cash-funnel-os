# Creator-to-Cash Funnel OS

Agent-native funnel routing for creator offers: intent capture, pricing path split, checkout handoff, and activation analytics.

## Run locally
```bash
pip install -r requirements.txt
uvicorn app.main:app --reload --port 8000
```

## Endpoints
- `/`
- `/docs-page`
- `/health`
- `/v1/public/config`
- `/llms.txt`
